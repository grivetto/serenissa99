# Serena Franceri - Portfolio Website (Beta 0.0.1)

Welcome to the digital portfolio and sports diary for Serena Franceri, an athlete and performer of aerial silks (tessuti aerei). This project highlights the grace, strength, and discipline required for the sport through a modern, "anti-gravity" inspired web design.

## Features
- **Hero Section:** Fullscreen layout with an impactful call-to-action ("Scopri il mio volo").
- **Gallery:** A modern CSS Grid layout displaying performance photos. Features fluid hover states.
- **Diario di Bordo:** A blog-style section for tracking technical progress, thoughts on choreography, and conditioning notes.
- **Contact:** A sleek, minimal contact form interface to get in touch.

## Tech Stack
- **HTML5:** Semantic HTML for robust structure and accessibility.
- **CSS3:** Custom styles utilizing CSS Grid, Flexbox, Variables, and Keyframes for fluid "antigravity" animations. Features dark mode styling with elegant Burgundy accents.
- **JavaScript (ES6):** Intersection Observers for elegant scroll-based entry animations, and a dynamic glowing mouse cursor tracker. No external UI frameworks are used, ensuring a fast and clean codebase.

## Setup & Deployment (GitHub Pages)
This project is optimized for deployment via GitHub Pages.

1. **Version Control:** Git is initialized on this directory. Initial version is mapped as Beta 0.0.1.
2. **Push to GitHub:**
   ```bash
   git remote add origin <YOUR_GITHUB_REPO_URL>
   git push -u origin main
   ```
3. **Deploy:** Go to the repository settings on GitHub, navigate to "Pages" on the sidebar, choose "Deploy from a branch", and select the `main` branch. Save the configuration and your website will be live.
